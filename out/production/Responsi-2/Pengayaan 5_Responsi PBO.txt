SOAL PENGAYAAN (Refleksi Polimorfisme)
Jawablah pertanyaan berikut:
1. Apa yang dimaksud dengan polimorfisme dalam OOP? Jelaskan bagaimana metode override bekerja.
2. Mengapa array bertipe DefaultMesin dapat menyimpan objek MesinMotor dan MesinTraktor?
3. Apa perbedaan override dan overload?
4. Apa keuntungan menggunakan polimorfisme dalam sistem inventaris seperti contoh di atas?
5. Bagaimana cara kerja dynamic dispatch pada Java saat memanggil method override?


1. Polimorfisme adalah kemampuan objek dari subclass untuk diperlakukan sebagai objek dari superclass. Method override memungkinkan subclass menyediakan implementasi berbeda dari method yang sama di parent class.

2. Array bertipe defaultMesin bisa menyimpan mesinMotor atau mesinTraktor karena kedua class tersebut mewarisi defaultMesin, jadi secara tipe mereka kompatibel.

3. Override: subclass mengganti implementasi method parent dengan signature sama. Overload: satu class memiliki beberapa method sama nama tapi berbeda parameter.

4. Keuntungan polimorfisme: kita bisa menyimpan dan memanipulasi berbagai jenis mesin dengan satu tipe array, memanggil method yang sesuai secara otomatis tanpa memeriksa tipe tiap objek.

5. Dynamic dispatch: saat memanggil method override, Java menentukan implementasi yang dipanggil pada runtime berdasarkan tipe objek sebenarnya, bukan tipe referensi.
